    002 Crypto Sell Sunshine With Litte Overpaid

    Description

    Успешный сценарий оплаты (продажи) криптовалюты с переплатой суммы ордера. Сумма переплаты - фиксированное значение, заданное в конфигурации.

    Preconditions

    • Check your API Key related to TARGET Merchant
    • Check your are use a correct environment
    • Prepare JSON body for Create order request
    • Grab info from service settings:
      • How many confirmations need to process an order on choosen currency
      • Amount needed for "litte overpaid"
    • Configure TARGET Merchant configuration like a limits, commissions, order timeouts, etc.
    • Subscribe HTTP callbacks for the topic DEPOSIT, ORDER

    Steps

    1. Create a CRYPTO_SELL order by call API Create order
      • HTTP 201
      • proper body content
      • check NEW order notification in ORDER topic
        • NEW order notification received
        • proper body content
      • check AWAIT_DEPOSIT order notification in ORDER topic
        • AWAIT_DEPOSIT order notification received
        • proper body content
    2. Grab amount.from and deposit.address. Make crypto deposit with exactly amount + amount needed for "litte overpaid". Wait for 0-confirmation (transaction in memory pool).
      • Check for second notification in ORDER topic
        • AWAIT_DEPOSIT order notification received
        • status = "PENDING"
        • state = "AWAIT_DEPOSIT"
        • paidStatus = "PAID"
        • deposit.transactions includes info of blockchain transaction
      • Check notification in DEPOSIT topic

        • TBD

      • Check for notifications in ORDER topic
        • AWAIT_DEPOSIT order notification received for each blockchain confirmations
        • status = "PENDING"
        • state = "AWAIT_DEPOSIT"
        • paidStatus = "PAID"
        • deposit.transactions includes info of blockchain transaction
      • Check notification in DEPOSIT topic

        • status = "PENDING"
        • state = "EXCHANGING"
        • paidStatus = "PAID"
        • deposit.transactions includes info of blockchain transaction
