    Terminology

    Instrument

    CryptoPay's Instrument defines kind of service. An instrument decribes how to process an order. Different instrument has own set of attributes according kind field.

    Note

    • Instrument defines applied commissions (may be overriden on in merchant's configuration)
    • Each merchant has own set of allowed instument

    Instrument kinds

    Kind Description
    CRYPTO_BUY Buy crypo asset (convert fiat to crypto)
    CRYPTO_SELL Sell crypo asset (convert crypto to fiat)
    CRYPTO_PAYMENT Payment for goods (buy goods by crypto)
