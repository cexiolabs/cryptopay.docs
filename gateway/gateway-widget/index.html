    Payment Widget

    Payment widget allows you guide you client over order complation within modal UI on you website. Here is the Widget Demo.

    Integration

    1. To use Payment Widget add cryptopay-widget.min.js to your website. 

      <script src="https://www.cryptopay.band/cryptopay-widget.min.js"></script>

    2. Create widget instance. 

      const widget = new CryptopayWidget({
  gatewayId: 'bd3476f9-57Bc-4517-b499-afb83e1538ес',  // Payment gateway Id
  elementId: 'cryptopay-widget', // Element to embed widget,
  onOpen: function () {}
  onClose: function () {}
});

    3. Call widget to present order information to client. 

      widget.show("<order id goes here>");

    Warning

    Do not rellay on order status from Widget. To make sure that order is completed you should check order status over Get order API method, or subscribe to API Callbacks and wait for corresponding callback.

    Example

    Following is HTML code which illustrates simple widget integration.

    <html>
    <head>
        <title>Widget demo</title>
    </head>
    <body>
    <script language="javascript">

        function onWidgetClosed(isCompleted) {

            if(isCompleted) {
                // Some actions to update presentation when 
                // user successfully paid the order
            }
            else {
                // Some actions to update presentation 
                // when order wasn't paid
            }
        }

        const widget = new CryptopayWidget({
            gatewayId: 'bd3476f9-57Bc-4517-b499-afb83e1538ес',
            elementId: 'cryptopay-widget',
            onClose: onWidgetClosed
        });

        widget.show(<order id goes here>)

    </script>
    </body>
    <div id="cryptopay-widget">
</html>
