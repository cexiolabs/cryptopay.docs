    @cexiolabs/сrypto.devel

    О проблеме

    Тестирование и отладка бизнес логики приложения через тестовые блокчейны очень неэффективна в плане временных затрат. Не работающий блокчейн, отсутствие фаусетов, ожидание включения транзакций в блок - все это пустая трата времени разработчика.

    Библиотека предоставляет набор инструментов для эффективной разработки в виртуальном окружении. Вы сможете получать виртуальные крипто-deposit-ы, управлять состоянием крипто-withdrawal-а, создавать исключительные ситуации всего в несколько кликов, не теряя фокуса в разрабатываемой бизнес логике.

    Состав:

    • CryptoWalletCurrencyReMapper - class обвертка вокруг экземпляра CryptoWallet выполняющая ре-мапинг валюты. Позволяет тестировать бизнес логику в основной валюте через альтернативную валюту. Например создавать ордера в ETH, а по факту работать с TRX блокчейном.
    • VirtualCryptoWallet - class Реализация интерфейса CryptoWallet в виде эмулятора виртуальных криптовалют. Решает проблемы тестирования и разработки бизнес логики (работа с фейковой криптой).
    • serve - variable Express Middleware которую можно добавить в ваш Express Application. Предоставляет возможность внешнего управления объектами VirtualCryptoWallet. Включается в себя:
      1. /api - API для управления виртуальной криптой (фейковые депозиты, список созданных адресов, балансы адресов). Экспортируется для автоматизированного тестирования и /web приложения.
      2. /web - Web-приложение для ручного управления (фейковые депозиты, список созданных адресов, балансы адресов).

    serve

    Express Middleware дает возможность добавить в ваш Express Application веб-интерфейс для управления виртуальными кошельками (инстансами VirtualCryptoWallet).

    import { serve } from "@cexiolabs/сrypto.devel";

const virtualCryptoWallet0: VirtualCryptoWallet = ...
const virtualCryptoWallet1: VirtualCryptoWallet = ...
const virtualCryptoWallet2: VirtualCryptoWallet = ...

expressApp.use("/some-path", serve(virtualCryptoWallet0, virtualCryptoWallet1, virtualCryptoWallet2));

// Open browser: http://your_express_app/some-path/web
// Or use API:   http://your_express_app/some-path/api

    List of addresses

    endDate ISO 8601 NO Date of create account, until EXCLUSIVE
    includeZero BOOLEAN NO Include zero balance accounts. Default: false.

    /web

    Web - приложение позволяет управлять виртуальными кошельками (инстансами VirtualCryptoWallet):

    1. Просматривать существующих список адресов
    2. Просматривать список deposit-ов
    3. Просматривать список withdrawal-ов
    4. Создавать deposit (тестирование сценариев оплаты, переплаты, недоплаты)
    5. Финализировать статус withdrawal-а (для тестирования сценариев отказа в выводе)
    6. Создавать withdrawal (для тестирования неожидаемых withdrawal)
    7. Создавать адрес (тестирование сценария с неожиданным адресом)
